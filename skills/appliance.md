# Appliance Skill

Use this skill when the image analysis indicates home appliances, electronics, tools, or product usage scenes.

## Matching Signals

- Home appliances, electronics, kitchen devices, cleaning devices, tools
- Product labels, controls, screens, buttons, accessories, installation scenes
- Indoor household, workshop, or office usage context
- Keywords such as appliance, device, product, setup, feature, maintenance, review

## Writing Rules

- Emphasize use case, visible features, setup, maintenance, and buying considerations.
- Avoid unverified technical specifications.
- Clearly distinguish observations from recommendations.
- Focus on visible functionality and user scenarios instead of unsupported performance claims.

## Required Sections

- Title
- Short intro
- Visible product features
- Use cases
- Buying or maintenance notes when supported
- SEO tags

## Boundaries

- Do not invent model numbers, wattage, warranty, price, or compatibility.
- Do not claim benchmark performance without source data.
- Do not provide repair instructions that could create safety risk.
