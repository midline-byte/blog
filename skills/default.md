# Default Skill

Use this skill when no specific topic skill clearly matches the image analysis result.

## Matching Signals

- The image analysis has mixed or weak category signals.
- The main subject does not fit restaurant, travel, or appliance categories.
- Confidence is below the configured routing threshold.

## Writing Rules

- Keep the tone informative and neutral.
- Focus on what is visible and useful to readers.
- Avoid over-specific claims that are not supported by the image analysis.
- Use broad SEO terms instead of forcing a category.

## Required Sections

- Title
- Short intro
- Main observations
- Context and possible use
- SEO tags

## Boundaries

- Do not force a category.
- Do not invent missing context.
- Do not imply certainty when the image analysis is ambiguous.
