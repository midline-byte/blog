# Project Setup

## Required Tools

- VSCode
- Git
- Node.js 22 LTS or newer
- Python 3.12 or newer
- Claude Code
- Codex CLI
- npm
- pip

## Initial Structure

The project uses these top-level directories:

- `docs`
- `input`
- `output`
- `prompts`
- `skills`
- `config`
- `logs`
- `src`
- `tests`

## Git

Run these commands after Git is available:

```bash
git init
git branch -M main
git add .
git commit -m "Initial Project Structure"
```
